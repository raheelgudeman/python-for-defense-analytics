# Python for Defense Analytics

## Overview
This repository contains beginner-to-intermediate Python analytics projects focused on defense, cyber, and military-relevant problem solving. The goal is to demonstrate how Python can support operational decision-making, data analysis, and capability development.

## Objectives
- Learn Python through real-world defense and military use cases
- Apply data analytics to operational and strategic problems
- Build a technical portfolio aligned with Army Cyber (17D) and analytics roles

## Skills Demonstrated
- Python fundamentals
- Data analysis with pandas
- Visualization with matplotlib
- Structured problem-solving
- Military and operational context application

## Repository Structure
- `basics/` – Python fundamentals and syntax
- `data/` – Sample datasets
- `notebooks/` – Jupyter notebooks with analysis
- `projects/` – Applied defense analytics mini-projects
- `docs/` – Explanations and write-ups
